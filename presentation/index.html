[OVERVIEW]

OMS is an AI-based surveillance system that monitors live camera feeds, detects people and objects in real time, supports tracking and face recognition, and provides a smart dashboard for monitoring.

01 PROBLEM STATEMENT

⚠️

Manual Fatigue

Traditional CCTV systems require continuous, passive human monitoring, leading to exhaustion.

📉

Human Error

Operators frequently miss critical, sudden, or fast-moving security events.

🖥️

Multi-Feed Overhead

Actively monitoring multiple feeds simultaneously is highly inefficient and expensive.

🔒

No Scene Analysis

Standard surveillance networks cannot automatically classify objects or detect intrusions.

02 OBJECTIVES OF OMS

Real-time Surveillance

Object Detection (YOLO)

[PIPELINE FLOW]

OMS receives video from a camera source, processes each frame, detects people/objects, performs recognition and tracking where needed, logs important events, and displays the results in the dashboard.

04 WORKING & MAIN MODULES

Core Modules

1. Camera Input Module

Decodes RTSP / USB video streams concurrently using OpenCV thread pool.

2. Object Detection Module

Passes images to YOLO for bounding box coordinates and object confidence.

3. Face Recognition Module

Matches localized facial embeddings with registered models.

4. Tracking & Logging Modules

Tracks movement trajectories and logs metadata to files.

Active Workflow Loop

  • 01
    Stream Capture:

    Continuously fetch raw video frames.

  • 02
    AI Analysis:

    Identify object boundaries and faces.

  • 03
